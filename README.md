# tailwebs# 🚀 TailWebs – React Landing Page

A modern, responsive landing page built using **React + TypeScript + Vite + Bootstrap**.  
This project focuses on clean UI, responsiveness, and reusable components.

---

## 📌 Tech Stack

- ⚛️ React (with TypeScript)
- ⚡ Vite (fast build tool)
- 🎨 Bootstrap 5 (responsive design)
- 🎯 AOS (Animate On Scroll)
- 💅 Custom CSS

---

## ✨ Features

- ✅ Fully responsive (Mobile + Tablet + Desktop)
- ✅ Smooth scrolling navigation
- ✅ Modern UI sections:
  - Hero Section
  - About Us
  - Areas of Work
  - Products & Solutions
  - Ecosystem
  - Our Platform
  - Resources
  - Footer
- ✅ Reusable components
- ✅ Clean layout & spacing
- ✅ Animations using AOS
